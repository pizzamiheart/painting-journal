# Painting Journal

A personal art exploration and journaling app. Search paintings across world-class museums, save favorites, and keep notes on the art that moves you.

## Features

- **Search** across Art Institute of Chicago, Rijksmuseum, and Metropolitan Museum of Art
- **Save** paintings to your collection with custom tags
- **Journal** your thoughts and observations on each piece
- **Painting of the Day** surfaces a random favorite to revisit
- **Spell-check** suggests corrections for artist name typos

## Quick Start

```bash
./run.sh
```

Opens automatically at `http://127.0.0.1:5000`

## Requirements

- Python 3.7+
- No API keys needed

## Stack

Flask / SQLite / Vanilla JS
